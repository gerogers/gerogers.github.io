Gomoku - Menu Items

Return to index. Leave help.

There are three drop down menus (four if you include Help)

  • Actions
  • Level
  • Options

    Actions

    The Actions menu drops down into 4 headings:

    1. New Game
    2. Load saved game
    3. Abort Load
    4. Save this game

    New Game

    This action clears the board and starts a new game with the current Sub-Goal list (see Sub-Goals).

    Load

    The Load action provides a mechanism for restoring previously saved situations. These can be:

    • a saved game history, such as a saved partially played game
    • a saved set of Sub-Goals, permitting a previously developed level of play to be restored
    • a full history containing both Sub-Goals and played moves
