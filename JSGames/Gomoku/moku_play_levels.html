Gomoku - Levels of play

Return to index. To leave help, click on the Help drop-down menu.

There are three default levels of play which may be set before the start of a game (see Level). The difference between these levels is the number of pre-defined sub-goals. By default, a single sub-goal is defined. As play progresses, the program may add its own sub-goals so that it, hopefully, won't make the same mistake again.

It is possible to save the current set of sub-goal descriptions and restore them at the start of a new session. (See Save and Load).
