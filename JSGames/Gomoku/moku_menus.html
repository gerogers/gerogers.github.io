Gomoku - Menus

Return to index. To leave help, click on the Help drop-down menu.

There are four drop down menus

  • Actions
  • Level
  • Options
  • Help

  • Actions

    • The Actions menu drops down into 4 headings:
    1. New Game
    2. Load saved game
    3. Abort Load
    4. Save this game

    1. New Game

      2. This action clears the board and starts a new game with the current Sub-Goal list (see Sub-Goals).

    2. Load saved game

      3. The Load menu provides a mechanism for restoring previously saved situations. The Load menu drops down to 3 sub-headings:
      • Load sub-goals
        Loads a saved set of Sub-Goals, permitting a previously developed level of play to be restored
      • Load saved game
        Loads a saved game history, such as a saved partially played game, leaving the play level (sub-goal list) unaffected
      • Load sub-goals and game
        Loads a full history containing both Sub-Goals and played moves

    3. Abort Load

      4. If you press Load saved game in error or if you can't find the saved file, you can use Actions/Abort Load to abort the Load action

    4. Save this game

      5. The Save menu provides a mechanism for saving the current game. The Save menu drops down to 3 sub-headings:
      • Save sub-Goals
        Displays the current set of sub-goals in a format suitable for saving to a local file using cut and paste
      • Save current game sequence
        Displays the current game history for saving to file
      • Save sub-goals and game
        Saves a full history containing both Sub-Goals and played moves

  • Level

    • The "level" menu offers three preset difficulty levels:
    1. Minimum
      2. This is the starting level and comprises a single sub-goal. The prime goal is to form 5 in a row. The first sub-goal describes a situation such that after playing a single piece, a pattern is left comprising either 4 in a row with both ends open (.xxxx.) or else two instances of 4 in a row, each of which has one end open ( 2 of *xxxx.). The sub-goal thus describes a situation from which a win may be forced in 2 moves.
    2. Medium
      3. The intermediate or medium level adds additional sub-goals covering all possible patterns from which a win can be forced in 3 moves.
    3. Tricky
      4. The preset collection of sub-goals set up under the "tricky" banner represents the sub-goals automatically generated after a good many games had been played by a skillful player.
